Kitchen Chaos Game

     

Kitchen Chaos is a high-energy, time-management cooking game where players step into the role of a chef managing a bustling kitchen. Players must prepare and serve a variety of dishes based on customer orders, all within a limited timeframe. With each level, the complexity increases, challenging players to balance multiple ingredients, recipes, and cooking processes while racing against the clock.

Perpetual Crusades

   

I created a two-player online dice-based board game with a theme loosely set around Dungeons & Dragons. It was developed as a submission for Penn Apps XXI, using ReactJS and CSS Grid Layouts for design and hosted on GitHub Pages.
